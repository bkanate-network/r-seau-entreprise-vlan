# 🏢 Réseau d'Entreprise Sécurisé — VLAN & Routage

## 📌 Description
Conception et déploiement d'un réseau d'entreprise sécurisé avec 
segmentation VLAN, routage inter-VLAN et supervision réseau.

## 🛠️ Technologies utilisées
- Cisco IOS (switches & routeurs)
- VLANs (802.1Q), STP, OSPF
- Cisco Packet Tracer 8.2.2

## 📁 Contenu du dépôt
| Dossier | Contenu |
|---------|---------|
| `schemas/` | Topologie réseau, plan d'adressage |
| `configs/` | Fichiers de configuration des équipements |
| `captures/` | Captures d'écran des tests et résultats |
